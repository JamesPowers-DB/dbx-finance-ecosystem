# DM Sans + DM Mono fonts

Place the following .ttf files here (download from Google Fonts or your internal asset store):

- dm-sans-regular.ttf         (400 normal)
- dm-sans-italic.ttf          (400 italic)
- dm-sans-medium.ttf          (500 normal)
- dm-sans-medium-italic.ttf   (500 italic)
- dm-sans-bold.ttf            (700 normal)
- dm-sans-bold-italic.ttf     (700 italic)
- dm-mono-light.ttf           (300 normal)
- dm-mono-regular.ttf         (400 normal)
- dm-mono-italic.ttf          (400 italic)
- dm-mono-medium.ttf          (500 normal)

These are referenced by @font-face in ../colors_and_type.css.
The fonts are not committed to the repo — add them before `npm run build`.
